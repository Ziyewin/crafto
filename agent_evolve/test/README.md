# 自生成工具执行智能体 (Self-Generating Tool Agent)

一个能**自己给自己写工具**的 AI 智能体。当原生大模型精度不足以可靠完成任务时，自动编写专用 Python 工具并立即调用执行。

驱动模型：DeepSeek（兼容 OpenAI API）

## 核心思想

传统 AI Agent 依赖预注册的工具箱（Calculator、WebSearch 等）。本框架换了个思路：

> **工具不是提前准备好的，而是按需生成的。**

遇到需要精确计算的数值任务时，Agent 会自动：
1. 判断「这个任务我的原生推理不可靠」
2. 编写一个精准的 Python 工具函数
3. 在本地执行该工具
4. 返回精确结果
5. 记住这个工具，下次同类任务直接复用

## 适用场景

- **高精度数学运算**：大数四则、方程求解、积分、三角函数、统计方差
- **批量运算**：列表计算、迭代处理、重复转换
- **数据格式校验**：JSON/CSV 解析、格式验证、复杂逻辑校验
- **任何容易算错的数值任务**

## 快速开始

### 1. 配置 API Key

```bash
export DEEPSEEK_API_KEY="sk-your-key-here"
```

或在 `config.py` 中直接修改 `DEEPSEEK_API_KEY` 的值。

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 运行

```bash
# 交互模式
python agent.py

# 单次查询
python agent.py "123456789 * 987654321 等于多少？"

# 查看版本
python agent.py --version
```

## 设计原则

- **零额外依赖**：生成的工具只使用 Python 内置标准库（math、decimal、json 等）
- **不编造结果**：所有数值结果必须通过工具函数本地计算得出
- **记忆复用**：会话内同类任务直接复用历史工具，不重复编码
- **结构化输出**：严格按【能力判定】【自研工具代码】【工具调用执行】【最终结果】分段

## 与 DeepSeek 的协作模式

LLM 承担两个角色：
1. **推理引擎**：判断任务可靠性、生成工具代码
2. **知识库**：概念解释、逻辑推理、文本分析

Python 运行时承担：
1. **精确计算**：数值运算、数据转换、批处理
2. **环境隔离**：工具在 subprocess 中执行，不污染主进程
3. **持久化**：工具代码和索引持久化到磁盘
