## 四、发送消息完整流程

### 4.1 前端 → 后端请求

```
用户输入文字 → 按 Enter
    │
    ▼
┌──────────────────────────────────────────────────────────────────┐
│ sendMessage()                     │ app.js │ 第321行             │
│                                                                    │
│  ① 获取输入: const text = $('#chat-input').value.trim()           │
│  ② 添加用户气泡: addMessage('user', text)                         │
│  ③ 添加 AI 占位气泡（"思考中..."动画）                             │
│  ④ 发起请求:                                                      │
│                                                                     │
│  const res = await fetch('/api/v1/chat/send', {                    │
│    method: 'POST',                                                  │
│    headers: getAuthHeaders(),   ← Authorization: Bearer <userId>   │
│    body: JSON.stringify({                                           │
│      session_id: state.activeSessionId || null,                     │
│      message: text,                                                 │
│      stream: false,                                                 │
│    }),                                                              │
│  })                                                                │
└────────────────────────────┬─────────────────────────────────────┘
                             │  HTTP POST /api/v1/chat/send
                             ▼
```

### 4.2 后端认证

```
┌──────────────────────────────────────────────────────────────────┐
│ AuthMiddleware.dispatch()      │ auth.py │ 第21行                │
│                                                                    │
│  ① path = "/api/v1/chat/send" → 非公开路由需要认证                │
│  ② auth_header = "Bearer <user_id>"                                │
│  ③ auth_header.startswith("Bearer ") → True                        │
│  ④ token = auth_header[7:]  → user_id 字符串                      │
│  ⑤ user_id = _resolve_token(token)                                 │
│     └─ db.query(User).filter_by(user_id=token).first()              │
│     └─ if user and user.is_active → return user.user_id             │
│  ⑥ request.state.user_id = user_id                                 │
│  ⑦ request.state.user_role = user.role                              │
│  ⑧ return await call_next(request)  ← 放行到路由                   │
└────────────────────────────┬─────────────────────────────────────┘
                             │
                             ▼
```

### 4.3 路由处理

```
┌──────────────────────────────────────────────────────────────────┐
│ POST /api/v1/chat/send          │ chat.py │ 第15行               │
│ async def send_message(req, request)                              │
│                                                                    │
│  user_id = request.state.user_id                                   │
│  user_role = request.state.user_role                               │
│                                                                    │
│  if not req.session_id:                                            │
│      session_mgr = SessionManager(user_id)                         │
│      session_id = await session_mgr.create_session()               │
│      │  └─ Session(user_id=..., session_id=uuid) → DB              │
│                                                                    │
│  agent = Agent(user_id, session_id, user_role)                     │
│  result = await agent.process_message(req.message)                 │
│                                                                    │
│  return result  ← 最终回复 JSON                                    │
└────────────────────────────┬─────────────────────────────────────┘
                             │
                             ▼
```

### 4.5 ReAct 循环（核心智能逻辑）

```
┌──────────────────────────────────────────────────────────────────┐
│ Agent._react_loop()                  │ agent.py │ 第108行        │
│ 最多迭代 5 轮                                                      │
│         1                                                            │
│  ┌─ for iteration in range(5): ──────────────────────────────┐    │
│  │                                                             │    │
│  │ ① 构建上下文:                                                │    │
│  │   messages = MemoryOrchestrator.build_context()              │    │
│  │   │  memory.py │ 第42行                                      │    │
│  │   ├─ [system] 注入摘要 + 长期记忆提示                         │    │
│  │   └─ [...]    短时消息（token 预算内滑动窗口）                │    │
│  │                                                               │    │
│  │ ② 获取可用工具:                                               │    │
│  │   tools = ToolRegistry.get_available_tools(                   │    │
│  │       include_code_gen=(user_role in ["advanced","admin"])    │    │
│  │   )                                                           │    │
│  │   │  registry.py │ 第20行                                     │    │
│  │   ├─ query_weather    → 预置                                 │    │
│  │   ├─ text_process     → 预置                                 │    │
│  │   ├─ date_calc        → 预置                                 │    │
│  │   └─ execute_python_code → 仅 advanced/admin                 │    │
│  │                                                               │    │
│  │ ③ 调用 DeepSeek API:                                          │    │
│  │   response = self.llm.chat(messages, tools=tools)             │    │
│  │   │  deepseek.py │ 第39行                                     │    │
│  │   ├─ POST https://api.deepseek.com/chat/completions           │    │
│  │   ├─ 返回 response.content + response.tool_calls              │    │
│  │   └─ response.finish_reason → "stop" 或 "tool_calls"          │    │
│  │                                                               │    │
│  │ ④ 判断是否有工具调用:                                          │    │
│  │   if response.tool_calls:  ← DeepSeek 决定调工具              │    │
│  │   │                                                           │    │
│  │   │ ⑤ 存 assistant 消息（一次，在循环外）                     │    │
│  │   │   add_assistant_message(..., tool_calls=raw_tool_calls)   │    │
│  │   │                                                           │    │
│  │   │ ⑥ 遍历执行每个工具:                                        │    │
│  │   │   for idx, tc in enumerate(tool_calls_in_response):       │    │
│  │   │     │                                                      │    │
│  │   │     │ tool_name = tc["name"]                               │    │
│  │   │     │ arguments = tc["arguments"]                          │    │
│  │   │     │ call_id = raw_tool_calls[idx]["id"]                  │    │
│  │   │     │                                                      │    │
│  │   │     │ result = await ToolRegistry.execute_tool(            │    │
│  │   │     │   tool_name, arguments, session_id                   │    │
│  │   │     │ )  │ registry.py │ 第65行                             │    │
│  │   │     │    │                                                    │    │
│  │   │     │    ├─ 如果是 execute_python_code:                     │    │
│  │   │     │    │   sandbox.client.execute_in_sandbox(code, uid)  │    │
│  │   │     │    │   │  sandbox/client.py │ 第15行                  │    │
│  │   │     │    │   ├─ scan_code(code)  │ code_scanner.py         │    │
│  │   │     │    │   │   └─ 18 条危险规则扫描                       │    │
│  │   │     │    │   └─ _mock_execute(code)                        │    │
│  │   │     │    │       └─ subprocess Python → 返回 stdout       │    │
│  │   │     │    │                                                  │    │
│  │   │     │    ├─ 如果是 query_weather / text_process / date_calc│    │
│  │   │     │    │   execute_preset_tool(name, args)               │    │
│  │   │     │    │   └─ tools/preset/weather.py::execute(city)     │    │
│  │   │     │    │                                                  │    │
│  │   │     │    └─ 如果是用户私有 Skill:                            │    │
│  │   │     │      SkillManager.execute_skill(skill_id, args)      │    │

│  │   │     │                                                      │    │
│  │   │     │ ⑦ 存 tool 响应消息:                                  │    │
│  │   │     │   add_tool_message(tool_result_str, call_id)         │    │
│  │   │     │   save_message("tool", ...)                          │    │

│  │                                                                  │    │
│  └──────────────────────────────────────────────────────────────┘  │
│                                                                     │
│ 达到 15 轮 → 返回结果摘要                                           │
└──────────────────────────────────────────────────────────────────┘
```

                                          │       │

                                          │       ├─ MCP Manager (按优先级排序)

                                          │

                                          │       │
                                          │    DeepSeekClient.chat()

                                          │
                                          ├─ ④ if finish_reason=="tool_calls"

                                          │    for each tool:
                                          │       │

                                          │       │   └─ MCP 优先 → fallback 本地
                                          │       │

                                          │       │   execute_preset_tool()
                                          │       │
                                          │       └─ 用户私有 Skill
                                          │           SkillManager.execute_skill()
                                          │
                                          ├─ ⑤ add_tool_message(result, call_id)
                                          ├─ ⑥ save_message("tool")
                                          ├─ ⑦ loop detection (3+ search → break)
                                          └─ ⑧ loop back to ① (max 15 times)
                                              │
                                         ⑧ 返回 (final_reply, tool_results, usage)

                                         ⑨ add_assistant_message(final_reply)

                                         ⑪ summarize_if_needed()

                                         return { reply, tool_calls, token_usage }
